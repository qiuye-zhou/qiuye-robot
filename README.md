# qiuye-robot
##### 功能

- 复读机
- 命令交互
- 通知(早晚安、欢迎新人)
- 简单翻译
- 艾特简单回复

------

```
>/ping

```

```
>/version

framework: oicq
```

```
>/tr-xxxxxxxxx

```
